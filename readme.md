## ❓ Frequently Asked Questions

### How do I reset progress?

Go to Settings → Reset Progress.

### How do I enable Mock Exam?

Go to Settings → Toggle "Mock Exam" on.

### Why are some topics missing?

Check if "All Topics" is enabled in Settings.
